# Compliance Language Rules

## Required Phrases
- "公开资料显示……"
- "根据来源 X 的报道……"
- "该记录尚需人工复核……"
- "该关系为模型抽取，置信度为……"
- "仅供研究参考，不构成事实认定、法律判断或投资建议。"

## Forbidden Phrases
- "此人腐败"
- "此人被收买"
- "此人一定受金主控制"
- "此人有黑料"
- "污点" / "劣迹" / "黑料"

## Controversy/Investigation Naming
- Always use "争议与调查记录 / Controversies & Investigations"
- Types: allegation, investigation, lawsuit, conviction, correction
- Never mark as factual conclusion without court/regulator/official confirmation
